## ✨ What's New

### 🌍 跨平台启动脚本

| OS | 临时启动 | 开机自启 | 依赖 |
|---|---|---|---|
| **Linux** (systemd) | `./start.sh` | `sudo ./install-service.sh` | `requirements.txt` |
| **macOS** (launchd) | `./start.sh` | `./install-service-macos.sh` | `requirements.txt` |
| **Windows** (Task Scheduler) | `start.bat` 或 `start.ps1` | DEPLOY.md 步骤 6 | `requirements-windows.txt` |

**`start.sh` 大升级：**
- 自动检测 OS（Linux/macOS/Windows）
- 兼容 macOS 默认 bash 3.2
- 4 级端口检查 fallback：`lsof` → `ss` → `netstat` → `/dev/tcp`
- 智能识别 systemd / launchd

**Windows PowerShell (`start.ps1`) 特色：**
- 彩色输出（红/绿/黄/青）
- 自动检测 `python` / `python3` / `py` 命令
- 错误信息友好提示
- 支持 `Get-Content ... -Wait` 实时日志

### 🔒 安全加固

**修复 3 处内网 IP 硬编码泄露**（commit `3e7cea9`）：

| 文件 | 位置 | 修复 |
|---|---|---|
| `backend/static/app.js` | line 1642 (分享文本) | `http://192.0.2.1:8001/` → `${window.location.origin}/` |
| `backend/static/index.html` | line 18 (`og:url` meta) | → `http://localhost:8001/` |
| `backend/static/index.html` | line 28 (`canonical` link) | → `http://localhost:8001/` |

**全仓敏感扫描结果**（36 个文件，12 种模式）：
- ✅ 0 命中
- 扫描范围：内网 IP (172.16/192.168/10.x) / GitHub tokens (ghp/gho/ghs/ghr/github_pat) / 邮箱 / password= / api_key= / secret= / 私钥

## 📦 Installation

### Linux

```bash
git clone https://github.com/PerfectZQ888/FIFA-WorldCup-2026-V2.git
cd FIFA-WorldCup-2026-V2
./start.sh                           # 临时启动
# 或开机自启:
sudo ./install-service.sh
```

访问 http://localhost:8001/

### macOS

```bash
git clone https://github.com/PerfectZQ888/FIFA-WorldCup-2026-V2.git
cd FIFA-WorldCup-2026-V2
./start.sh                           # 临时启动 (兼容 bash 3.2)
# 或开机自启:
./install-service-macos.sh
```

> 需要 Python 3.11+：`brew install python@3.11`

### Windows

```cmd
git clone https://github.com/PerfectZQ888/FIFA-WorldCup-2026-V2.git
cd FIFA-WorldCup-2026-V2
start.bat
```

或 PowerShell（推荐）：
```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

> 开机自启见 DEPLOY.md "Windows 10/11 部署" 章节（Task Scheduler 配置）

## 🔄 Upgrade from 之前版本

**无破坏性变更**。如果你之前已经手动部署了 V2：

```bash
cd /path/to/FIFA-WorldCup-2026-V2
git pull origin main
# 重启服务:
sudo systemctl restart wc2026-v2     # Linux
# 或
launchctl unload ~/Library/LaunchAgents/com.wc2026-v2.plist
launchctl load -w ~/Library/LaunchAgents/com.wc2026-v2.plist   # macOS
```

## 🐛 Known Issues

- Windows 上 `uvloop` 不可用 → 已在 `requirements-windows.txt` 移除，性能略低但功能完整
- Windows 自启需手动配置 Task Scheduler（DEPLOY.md 步骤 6）
- macOS 多用户场景需每个用户单独运行 `install-service-macos.sh`
